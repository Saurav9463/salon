## FIX 1 — Navbar (MOST CRITICAL)

The mobile navbar menu is broken — it overlaps content and never closes.

In src/components/layout/Navbar.tsx, implement a proper mobile nav:

### State
Add this state at the top of the Navbar component:
  const [isOpen, setIsOpen] = useState(false);

### Hamburger button
On mobile, show a hamburger/close toggle button:
  <button
    className="md:hidden flex flex-col gap-1.5 p-2 z-50 relative"
    onClick={() => setIsOpen(!isOpen)}
    aria-label="Toggle menu"
  >
    {isOpen ? (
      <X size={24} className="text-foreground" />
    ) : (
      <>
        <span className="w-6 h-px bg-foreground block" />
        <span className="w-6 h-px bg-primary block" />
        <span className="w-4 h-px bg-foreground block" />
      </>
    )}
  </button>

### Desktop nav links
Wrap existing nav links so they ONLY show on md and above:
  <nav className="hidden md:flex items-center gap-8">
    {/* existing nav links */}
  </nav>
  <div className="hidden md:block">
    {/* existing BOOK NOW button */}
  </div>

### Mobile dropdown menu
Add a full-screen mobile menu overlay BELOW the navbar bar:
  {isOpen && (
    <div className="fixed inset-0 top-[64px] bg-background z-40 flex flex-col 
                    items-center justify-center gap-10 md:hidden">
      {["Home", "Services", "Team", "Gallery", "Contact"].map((item) => (
        <Link
          key={item}
          href={item === "Home" ? "/" : `/${item.toLowerCase()}`}
          onClick={() => setIsOpen(false)}
          className="text-3xl font-serif text-foreground hover:text-primary 
                     transition-colors"
        >
          {item}
        </Link>
      ))}
      <Link
        href="/book"
        onClick={() => setIsOpen(false)}
        className="ghost-btn-gold px-10 py-4 text-lg mt-4"
      >
        Book Appointment
      </Link>
    </div>
  )}

### Close menu on route change
Add this effect:
  useEffect(() => {
    setIsOpen(false);
  }, [location]);

### Prevent body scroll when menu open
Add this effect:
  useEffect(() => {
    document.body.style.overflow = isOpen ? "hidden" : "";
    return () => { document.body.style.overflow = ""; };
  }, [isOpen]);
