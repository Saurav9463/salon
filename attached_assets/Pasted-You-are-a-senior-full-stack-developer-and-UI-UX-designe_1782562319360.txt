## 🎨 DESIGN DIRECTION

**Theme:** Dark luxury. Think high-end editorial — like a blend between a fashion magazine and a premium spa brand.

**Color Palette:**
- Background: #0A0A0A (true near-black, not gray)
- Surface cards: #111111
- Borders: #1E1E1E
- Primary accent: #C9A96E (warm champagne gold)
- Secondary accent: #8B6F47 (deeper bronze)
- Text primary: #F5F0EB (warm off-white)
- Text muted: #6B6560

**Typography:**
- Display/headings: `Cormorant Garamond` (Google Font) — elegant, editorial, high-fashion
- Body: `DM Sans` — clean, modern, highly readable
- Utility/labels: `DM Mono` — for prices, tags, small metadata

**Aesthetic rules:**
- Large whitespace — sections must breathe
- Thin gold dividers and lines as decorative elements
- Subtle texture overlays (CSS noise or grain) on hero sections
- No generic stock photo placeholders — use beautiful CSS gradient cards where images are missing
- Zero rounded corners on cards (sharp, editorial feel)
- Hover states: gold underline animations, not background color fills
- Avoid gradients on buttons — use outlined/ghost styles in gold
- Every font size must follow a clear typographic scale

## 🗄️ SUPABASE SCHEMA

Create and use these tables:

```sql
-- Bookings
create table bookings (
  id uuid primary key default gen_random_uuid(),
  client_name text,
  client_email text,
  client_phone text,
  service_id uuid references services(id),
  stylist_id uuid references team(id),
  appointment_date date,
  appointment_time time,
  status text default 'Pending',
  created_at timestamptz default now()
);

-- Services
create table services (
  id uuid primary key default gen_random_uuid(),
  name text,
  category text,
  description text,
  duration_minutes int,
  price numeric,
  active boolean default true
);

-- Team
create table team (
  id uuid primary key default gen_random_uuid(),
  name text,
  role text,
  bio text,
  speciality text,
  photo_url text,
  years_experience int,
  active boolean default true
);

-- Gallery
create table gallery (
  id uuid primary key default gen_random_uuid(),
  image_url text,
  category text,
  created_at timestamptz default now()
);

-- Messages
create table messages (
  id uuid primary key default gen_random_uuid(),
  name text,
  email text,
  message text,
  read boolean default false,
  created_at timestamptz default now()
);

-- Settings
create table settings (
  key text primary key,
  value text
);
```
